# Changelog

All notable changes to this project are documented in this file.

## 2026-04-03

### 1.0.1-r1

- Improved top navigation and dropdown contrast in the `defdo` theme.
- Added a packaged theme version marker at:
  - `/usr/share/luci-theme-defdo/VERSION`
- Published the `v1.0.1-r1` release artifact:
  - `luci-theme-defdo_1.0.1-r1_all.ipk`

### 1.0.0-r1

- Created the standalone `luci-theme-defdo` repository and package layout.
- Switched package versioning to the standard OpenWrt `PKG_VERSION` + `PKG_RELEASE` flow.
- Aligned release tags and workflow logic with the actual built `.ipk` version.
- Published the first packaged release:
  - `luci-theme-defdo_1.0.0-r1_all.ipk`
